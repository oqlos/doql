# Mask Inspection Manager — Desktop (Tauri)

Generated by `doql build`. Tauri 1.5 shell wrapping the web frontend.

## Prerequisites

- Rust toolchain: <https://rustup.rs>
- Node 20+ (for the web frontend)

## Quick start

```bash
cd build/desktop
npm install
npm run dev
```

## Build installer

```bash
npm run build
# Artifacts in src-tauri/target/release/bundle/
```
