# Mask Inspection Manager — Mobile (PWA)

Generated by `doql build`. Installable PWA with offline support.

## Quick start

```bash
cd build/mobile
# Serve over HTTP (service worker requires http/https, not file://)
python3 -m http.server 8090
# Open http://localhost:8090/ in Chrome/Firefox
# On Android: 'Install app' from the address bar menu.
```

## Connecting to the API

By default the PWA calls `/api/v1/...` on the same origin. To point
at a different API (e.g. during development), set before loading app.js:

```html
<script>window.DOQL_API_BASE = 'http://localhost:8000';</script>
```

## Production

Serve `build/mobile/` behind the same Traefik / nginx that hosts the API,
so `/api/v1` is same-origin and offline cache works as expected.
