# Notes — Desktop (Tauri v2)

Generated by `doql build`. Tauri 2.x shell wrapping the web frontend.

## Prerequisites

- **Rust** toolchain: <https://rustup.rs>
- **Node 20+** (for the web frontend)
- **System libraries** (Linux):
  ```bash
  sudo apt install -y \
      libwebkit2gtk-4.1-dev libsoup-3.0-dev \
      libgtk-3-dev libayatana-appindicator3-dev \
      librsvg2-dev libssl-dev pkg-config build-essential
  ```

## Quick start

```bash
cd build/desktop
npm install
npm run dev
```

On first run cargo downloads ~300 deps (~3-5 min). Subsequent runs use the cache.

## Build installer

To produce `.deb`/`.AppImage`/`.dmg`/`.msi`, first add icons to `src-tauri/icons/`
and set `bundle.active = true` + populate `bundle.icon` in `tauri.conf.json`,
then:

```bash
npm run build
# Artifacts in src-tauri/target/release/bundle/
```
